# 📌 Project Overview

Digital marketing teams monitor campaign performance to understand how advertising budgets are being spent and which campaigns generate the best business outcomes.

In this project, a raw advertising campaign dataset was cleaned, transformed, analyzed, and visualized entirely in Microsoft Excel. The final deliverable is an interactive dashboard that enables users to monitor campaign performance using key digital marketing metrics and dynamic filters.

---

# 🎯 Project Objectives

- Prepare raw campaign data for analysis
- Clean inconsistent and incomplete records
- Calculate important marketing KPIs
- Build Pivot Tables for business analysis
- Design an interactive Excel dashboard
- Generate insights to support campaign optimization

---

# 📂 Dataset Information

- **Dataset Type:** Mock Digital Advertising Campaign Data
- **Reporting Period:** November 2024
- **Records:** 2,600 Campaign Records
- **Tool Used:** Microsoft Excel

# 🧹 Data Cleaning & Transformation

Before analysis, the raw dataset was cleaned and standardized using Microsoft Excel.

### 1. Structured the Dataset

- Converted the raw dataset into an Excel Table for dynamic referencing and easier data management.

---

### 2. Duplicate Check

- Verified the dataset for duplicate records.
- Confirmed that every Ad_ID was unique.

---

### 3. Standardized Text Values

Cleaned inconsistent text values using **Find & Replace** and manual verification.

Columns standardized:

- Campaign Name
- Device
- Location

Example:

```
DataAnalyticsCourse
Data Anlytics Corse
Data Anlaytics Corse
Data Analytcis Course
```

↓

```
Data Analytics Course
```

---

### 4. Standardized Date Format

The dataset contained multiple date formats such as:

```
16/11/2024
20-11-2024
```

These were converted into a consistent date format using:

- ISNUMBER()
- DATEVALUE()
- SUBSTITUTE()
- TRIM()

---

### 5. Missing Value Analysis

Reviewed the dataset for missing values in:

- Clicks
- Impressions
- Cost
- Leads
- Conversions
- Conversion Rate
- Sale Amount

---

### 6. Filled Missing Conversion Rate

Calculated missing Conversion Rate values using existing Clicks and Conversions wherever sufficient information was available.

---

### 7. Filled Missing Clicks & Conversions

Calculated missing Clicks and Conversions using related available metrics instead of replacing values manually, maintaining consistency across the dataset.

---

### 8. Created Calculated Columns

Additional columns were created to support KPI calculations and dashboard analysis.

| Column | Description |
|---------|-------------|
| CTR | Click Through Rate |
| CPC | Cost Per Click |
| CPL | Cost Per Leads |
| ROAS | Return On Ad Spend |
| Campaign Status | Campaign Performance Classification |
| Day | Day of Month |
| Weekday | Weekday Name |
